# MVL Benchmark Generator

Generate Multi-Valued Logic (MVL) ALU Benchmarks using Large Language Models.

## Features

- Support for multiple LLM providers (Gemini, OpenAI, Groq, DeepSeek, Mistral, Qwen, Together AI)
- Generate MVL ALU code in C, Python, Verilog, VHDL
- Support K-values from 2 to 7
- Support bitwidths: 8, 10, 12, 14 trits
- Built-in simulation runner (GCC, Python, Icarus Verilog)

## Research Paper

This project is part of the research for ISMVL 2026:
> "LLM-based Generation of High-Level Benchmarks for MVL Designs"

## Authors

- Prof. Dr. Rolf Drechsler
- Qian Liu 


## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set environment variables for API keys
4. Run: `python web/app.py`
